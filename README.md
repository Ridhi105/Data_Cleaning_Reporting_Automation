# Data Cleaning & Reporting Automation

## Project Overview

Data Cleaning & Reporting Automation is a Python-based project designed to automate the process of cleaning raw datasets and generating analytical reports. The project handles missing values, removes duplicate records, standardizes data formats, and creates visual reports to improve data quality and reporting efficiency.

This project demonstrates practical applications of data preprocessing, automation, and data visualization, which are essential skills in data analytics and business intelligence.

---

## Objectives

* Automate data cleaning tasks.
* Handle missing and inconsistent data.
* Remove duplicate records.
* Generate summary reports automatically.
* Create visualizations for data analysis.
* Improve reporting efficiency and data quality.

---

## Dataset

Dataset Used: Netflix Movies and TV Shows Dataset

The dataset contains information about Netflix content, including:

* Title
* Type (Movie/TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre
